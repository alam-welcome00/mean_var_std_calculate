## Mean-Variance-Standard Deviation Calculator
#### 📌 Project Overview

This project reshapes a list of 9 numerical values into a 3×3 matrix using NumPy and calculates key statistics including:

Mean
Variance
Standard Deviation
Maximum
Minimum
Sum

The calculations are provided across columns, rows, and the overall matrix.

This project is part of the Data Analysis with Python curriculum on FreeCodeCamp
.

#### ⚡ Features

Accepts user input (9 numbers separated by spaces).

Converts the input into a 3x3 NumPy array.

Computes descriptive statistics for each row, each column, and the entire matrix.

Outputs results in a structured dictionary format.
